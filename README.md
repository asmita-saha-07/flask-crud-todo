# Flask ToDo App

A simple and efficient task management web application built using Flask, SQLAlchemy, and Bootstrap. This project demonstrates the implementation of full CRUD (Create, Read, Update, Delete) operations in a Flask-based web application.

## Features

* Create new tasks
* View all existing tasks
* Update task details
* Delete completed or unwanted tasks
* SQLite database integration
* Clean and responsive Bootstrap interface

## Tech Stack

* Python
* Flask
* SQLAlchemy
* Bootstrap 5
* Jinja2 Templates

## Project Structure

```
todoapp.py
templates/
├── home.html
└── update.html
todo.db
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/asmita-saha-07/flask-todo-app.git
```

2. Navigate to the project folder

```bash
cd flask-todo-app
```

3. Install dependencies

```bash
pip install flask flask-sqlalchemy
```

4. Run the application

```bash
python todoapp.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
